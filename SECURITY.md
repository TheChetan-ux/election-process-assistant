# Security Policy

## Supported Versions

We only support the latest version of VoteWise. Please ensure you are running the most recent release for all security patches.

## Reporting a Vulnerability

We take the security of the election process seriously. If you find a vulnerability, please do not disclose it publicly. 

Instead, please report it via the following process:
1. Open a private issue on this repository (if supported) or contact the maintainers.
2. Provide a detailed summary of the vulnerability.
3. We will acknowledge your report within 48 hours and provide a fix as soon as possible.

## Responsible AI Disclosure

This assistant uses Google Gemini. While we have implemented strict safety filters, AI can sometimes produce unexpected results. We encourage users to verify critical election information at [eci.gov.in](https://eci.gov.in).
