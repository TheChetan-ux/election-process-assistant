# Contributing to VoteWise

We love your input! We want to make contributing to VoteWise as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features

## Our Development Process

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints (`flake8`).

## Style Guide

- We use **PEP 8** for Python.
- We use **Google Style** docstrings.
- We use **Type Hints** for all function signatures.

Thank you for helping empower democracy through technology!
